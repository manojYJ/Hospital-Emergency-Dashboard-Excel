# 🏥 Hospital Emergency Room Dashboard | Healthcare Data Analytics (Excel)

## 📌 Project Overview
I developed an interactive **Hospital Emergency Room Dashboard** in Excel to analyze and monitor key operational metrics within an ER environment.  
This project enhanced my understanding of how Excel can be leveraged for healthcare analytics, supporting efficiency, patient care, and informed decision-making.

## 🎯 Key Analytical Areas Covered
- **Patient Admission Status**: Analysis of admitted vs. non-admitted patients  
- **Patient Age Distribution**: Grouping by age categories to identify demand patterns  
- **Timeliness of Care**: Measurement of patients attended within the 30-minute benchmark  
- **Gender Analysis**: Patient distribution by gender  
- **Department Referrals**: Identification of departments receiving the highest referrals  

## 📊 Key KPIs Implemented
- **Number of Patients**:  
  Total ER visits tracked with trend analysis using Excel charts and sparklines to identify peak and seasonal patterns  
- **Average Wait Time**:  
  Average time patients waited before consultation, highlighting operational bottlenecks  
- **Patient Satisfaction Score**:  
  Daily average satisfaction scores visualized to monitor service quality and correlate with patient volume and wait times  

## 🛠️ Tools & Techniques
- **Excel Functions**: Pivot tables, formulas, conditional formatting  
- **Data Cleaning**: Sorting, filtering, and structured tables  
- **Visualization**: KPI cards, bar charts, pie charts, sparklines, and departmental breakdowns  
- **Interactivity**: Slicers and filters for dynamic analysis
